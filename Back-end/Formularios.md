#### Estados
Permiten condicionar el comportamiento de un elemento de formulario a partir del estado otro elemento.
```php
$form['boys'] = [
  '#type' => 'checkbox',
  '#title' => $this->t('Do you have boys?'),
];

$form['boys_nomber'] = [
  '#type' => 'textfield',
  '#title' => $this->t('How many boys do you have?'),
  '#states' => [
    'visible' => [
      'input[name="boys"]' => ['checked' => TRUE],
    ]
  ]
];

```
Estados: 
- visible
- invisible
- enabled
- disabled
- required
- optional
- checked
- unchecked
- collapsed

Detonantes:
- checked
- empty
- filled
- unchecked
- expanded
- collapsed
- value
